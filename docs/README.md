# Tracy's Pet

Tracy got an angry pet to remind you of your failures

## Content

- [Setup](#setup)
- [Usage](#usage)

## Setup

Install with [Composer](https://getcomposer.org)

```sh
composer require orisai/tracys-pet
```

Register extension

```neon
extensions:
	orisai.tracysPet: OriNette\TracysPet\DI\TracysPetExtension
```
